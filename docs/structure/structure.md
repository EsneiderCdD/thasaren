# Estructuras de Datos y Colecciones - Checklist de Progreso

| Estado | Categoría | Concepto |
| :---: | :--- | :--- |
| ✅ | **Objetos** | Declaración Literal (Propiedades y valores) |
| ✅ | **Objetos** | Acceso a propiedades (Notación de punto `.`) |
| ✅ | **Objetos** | Acceso dinámico (Notación de corchetes `[]`) |
| ✅ | **Objetos** | Mutabilidad (Referencia vs Valor) |
| ✅ | **Objetos** | Destructuring (Extracción de propiedades) |
| ✅ | **Objetos** | Spread Operator `...` (Copia superficial y merge) |
| [ ] | **Objetos** | Propiedades abreviadas (Shorthand properties) |
| [ ] | **Objetos** | Nombres de propiedades computados |
| [ ] | **Objetos** | Keyword `this` y contexto básico |
| [ ] | **Objetos** | Métodos estáticos: `Object.keys`, `values`, `entries` |
| [ ] | **Objetos** | `Object.freeze` vs `Object.seal` |
| [ ] | **Objetos** | Optional Chaining `?.` con métodos |
| ✅ | **Arrays** | Declaración y propiedad `.length` |
| ✅ | **Arrays** | Acceso por índice y límites |
| ✅ | **Arrays** | Métodos mutables (Pila/Cola: `push`, `pop`, `shift`, `unshift`) |
| [ ] | **Arrays** | Métodos mutables avanzados (`splice`, `sort`, `reverse`) |
| ✅ | **Arrays** | Métodos inmutables de iteración (`map`, `filter`, `find`) |
| [ ] | **Arrays** | Métodos de reducción (`reduce`) |
| [ ] | **Arrays** | Métodos de verificación (`includes`, `some`, `every`) |
| [ ] | **Arrays** | Métodos de índice (`indexOf`, `findIndex`) |
| [ ] | **Arrays** | Aplanamiento (`flat`, `flatMap`) |
| ✅ | **Arrays** | Destructuring de arrays (Posicional) |
| ✅ | **Arrays** | Spread Operator con Arrays (Concatenar/Expandir) |
| [ ] | **Map & Set** | Estructura `Map` (Claves de cualquier tipo) |
| [ ] | **Map & Set** | Estructura `Set` (Valores únicos) |
| [ ] | **Map & Set** | Iteración sobre mapas y conjuntos |
| [ ] | **Map & Set** | Conversión entre Objetos/Arrays y Map/Set |
| [ ] | **JSON** | Formato JSON vs Objeto JS |
| [ ] | **JSON** | Serialización (`JSON.stringify`) |
| [ ] | **JSON** | Deserialización (`JSON.parse`) |
