# Roadmap — Biblioteca Personal de JavaScript

| Módulo | Dominio | Subtemas principales | Tipo de práctica diaria | Resultado que desarrollas |
|--------|----------|----------------------|--------------------------|---------------------------|
| 1 | Lógica pura | Condicionales reales, funciones puras, control de flujo, closures, validaciones, reglas de negocio | Sistemas pequeños sin DOM (energía, puntos, inventario, validadores) | Pensamiento estructurado y diseño de reglas |
| 2 | Estructuras de datos | Arrays avanzados, objetos complejos, map/filter/reduce profundo, Sets, Maps, normalización | Transformar, agrupar y buscar datos en estructuras simuladas | Modelado mental de datos |
| 3 | Estado y arquitectura | Separación estado / lógica / render, inmutabilidad, composición | Mini motores (motor de reglas, gestor de estado simple) | Diseño interno tipo mini-framework |
| 4 | DOM | Selección, creación, eliminación, render dinámico, sincronización estado → vista | Listas dinámicas, toggles complejos, mini render manual | Entender qué hace React realmente |
| 5 | Eventos | Bubbling, delegación, teclado, mouse, eventos personalizados | Sistemas interactivos (atajos, drag simple, teclado tipo juego) | Control fino de interacción |
| 6 | Tiempo y asincronía | setTimeout, setInterval, Promesas, async/await, manejo de errores | Simuladores de carga, secuencias temporales, retry automático | Control del flujo temporal |
| 7 | Fetch y red | GET, POST, manejo de errores HTTP, paralelismo, AbortController | Consumo real de API, sistema de carga y estados | Comunicación real con backend |
| 8 | Storage | localStorage, sessionStorage, persistencia de estado | Guardar progreso, configuración, sesión simple | Persistencia en frontend |
| 9 | Workers | Web Workers, postMessage, separación de hilo | Cálculos pesados sin bloquear UI | Comprensión del hilo principal |
| 10 | Canvas | Coordenadas, loop de render, animación manual, colisiones básicas | Mini motor gráfico, movimiento controlado | Control matemático del render |
| 11 | Comunicación en tiempo real | WebSockets básicos | Chat mínimo o notificador en tiempo real | Flujo bidireccional continuo |
| 12 | Performance | Debounce, throttle, memoización, profiling básico | Optimizar algo que ya hiciste antes | Mentalidad profesional |
